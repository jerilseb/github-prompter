Your token will be securely stored in Browser storage.
How to get a GitHub Personal Access Token (PAT)
  1. Go to GitHub Settings > Developer settings > Personal access tokens.
  2. Click "Generate new token" (select "tokens (classic)").
  3. Give your token a descriptive name (e.g., "GitHub Prompter Extension").
  4. Set an expiration date (recommended).
  5. Select the `repo` scope (Full control of private repositories). This is needed to access private repository contents and avoid rate limits on public ones.
  6. Click "Generate token".
  7. Copy the token immediately! You won't be able to see it again.
  8. Paste the copied token into the input field above and click "Save Token".

For more details, see the GitHub documentation.
