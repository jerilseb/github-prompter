Github Prompter Settings

How to get a GitHub Personal Access Token (PAT)
  1. Go to GitHub Settings > Developer settings > Personal access tokens.
  2. Click "Generate new token" (select "tokens (classic)").
  3. Give your token a descriptive name (e.g., "GitHub Prompter Extension").
  4. Set an expiration date (recommended).
  5. If you only want to access public repositories, no need to select any scope.
  6. If you want to access private repositories, select the `repo` scope (Full control of private repositories).
  7. Click "Generate token".
  8. Copy the token immediately! You won't be able to see it again.
  9. Paste the copied token into the input field above and click "Save Token".

For more details, see the GitHub documentation.
